== Description ==
Relais 2FA allows you to improve your site by adding an authentication method.
The principlee is simple: enter your email and then scan the QR code shown. No password needed.

== Installation ==
1) Download and activate the plugin.
2) Download the mobile app Relais 2FA from your store (android & ios soon).
Android: https://play.google.com/store/apps/details?id=fr.mobisoft.relais
IOS: soon
3) Pair your device by scanning the QR code in the administration panel.
